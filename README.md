# 1. 프로젝트 설계
## 목적
- 리액트 기반 SPA 프로젝트 공부
- 파편화되어있는 라이브러리 소스를 한 데 모아 가이드화
- 공개 포트폴리오 사이트 구축
- 포스트 가능한 블로그 기능 탑재
